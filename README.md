# java-sprint2-hw
## Это репозиторий проекта "Трекер Задач"


Приложение **умеет**:
1. Создавать задачи трех типов.
2. Обновлять задачи разных типов.
3. Получать список задач.
4. Получать задачу по ID
5. Удалять задачу по ID
6. Удалять все задачи

    }
